BTSI Lab Systems Integration : Releasing PATAC Solution

Created by James Jezewski on Jan 14, 2021

Approximately once every quarter, we need to send a copy of the latest master branch of solution_common to PATAC.

To export the solution:

  1. Run itestX.X\solution_common\Sanitize and Export.bat

  2. This will create a new folder: itestX.X\Solution_Common_Export_Ready
  3. Manually copy the schedules folder from Solution_Common to Solution_Common_Export_Ready

  4. **VERY IMPORTANT** Remove any schedules relevant to: H5, BEV3 RoW, and ET1B 

    These programs are GM confidential and information related to these programs should not be shared  

  5. Replace PagerDuty API Keys(PXshx37gov4foqvRstBr) and (R011B9NQLE5H4VYZSEN6RAVGZ0GKLFB3) with ****** in these files:
    1. Solution_Common_Export_Ready\EXECUTE\PagerDutyNotify.py
    2.  Solution_Common_Export_Ready\EXECUTE\PagerDuty\UpdateStandOwner.py
  6. zip solution and send to test engineer responsible for sending the solution to PATAC
