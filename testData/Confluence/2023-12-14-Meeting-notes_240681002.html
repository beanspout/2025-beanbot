BCIC Updates
  • We had a PXI controller that had a hard drive die. It's going back to NI for repairs. It would run for 4-8 weeks and then disconnect from Veristand while on-test. Eventually it refused to boot at all. We are going to pull it and ship it back to NI
  • Collin submitted a bug report to NI about how the Shared test files work in BTS. The BTS server stores the shared files in a weird way that overwrites old versions rather than keeping an archive of each version (the documentation says it should never delete old files), which means that tests will always deploy with the latest version of the shared files, even if they are supposed to use an older version. The NI BTS team is investigating this and will decide if it requires an immediate patch or if it will be fixed in a future release of BTS.
