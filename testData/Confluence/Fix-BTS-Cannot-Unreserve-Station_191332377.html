BTSI Lab Systems Integration : Fix BTS Cannot Unreserve Station

Created by Collin Winans on Mar 28, 2023

Sometimes, BTS gets confused and thinks a test site is reserved even though it is not in use, and unchecking the box doesn't unreserve the test site. When you try to start a test, it will report that another test plan is already running even though that test plan is running on another test site.


There is an issue with how the copy/paste setup was done during the initial setup. BTS gets confused and thinks the stands are the same.



Please navigate to C:\ProgramData\National Instruments\Battery Test System and delete the .teststationid file. BTS should create a new one after you delete it:


Now the station should not be reserved in BTS
