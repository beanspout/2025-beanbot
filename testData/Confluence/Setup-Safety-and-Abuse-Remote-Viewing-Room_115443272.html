BTSI Lab Systems Integration : Setup Safety and Abuse Remote Viewing Room

Created by Chelsea Flattery, last updated on Feb 23, 2022

The conference room has two PCs setup for remote viewing of TC89A and the FLIR camera (TC89B).  These PCs have view-only access to see the Safety and Abuse screens.

Step-by-step guide

How to connect Remove Viewing PCs to Safety and Abuse PCs

To turn them on, use the touchpad on the desk and select Send To <Left/Right> LCD Display under the Screen Select menu then use the remote (in the drawer under the screens) to change the source to HDMI.  Repeat for the other Display.  Send to both doesn't seem to work reliably.

  1. uswtlbd0035083 is the remote viewing PC for TC89A
  2. uswtlbd0061697 is the remove viewing PC for the FLIR camera PC (TC89B)
  3. Login to the PCs in the remote viewing conference room using VEHTESTFIDLABMINDER and the normal lab password
  4. Open UltraVNC Viewer
  5. The server should automatically populate
    TC89A = 10.210.196.44
    FLIR PC = 10.210.196.12
  6. The Direct radio button should be selected
  7. Click Show Options to expand the settings menu
  8. In the Encoders menu, confirm that Auto select the best settings is checked
  9. Click Connect
  10. For normal view-only viewers Z:\_LabPasswords.ini
    1. This account should be used at all times, unless features need to be changed
  11. For administrative tasks Z:\Software Team\_LSIESystems.ini


A bulk of the settings are in the UltraVNC Settings on the host machines.
