BTSI Lab Systems Integration : SystemLink Custom Devices

Created by Collin Winans, last updated on Apr 12, 2024

Note: this might get moved to a Bitbucket repo in the future, for now I'm stashing the most recent version from NI here.


Any PC running a BTS test for BCIC will need the BCIC custom devices installed. The lab PCs should already have this (installed by UTAC during comissioning). I find it's useful to have a fake "test site" set up on my laptop for script development purposes.


The nipkg file should open with NI Package Manager and install

v2.0.0.16

bcic-custom-devices_2.0.0.16_windows_x64.nipkg


Put the lookup table plugin into C:\Users\Public\Documents\National Instruments\Battery Test System\Compiler\Plugins

LookupTableInterface.lvlibp
