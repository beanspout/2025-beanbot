BTSI Lab Systems Integration : Update Vehicle Spy License Files

Created by Chelsea Flattery, last updated on Feb 23, 2022

You notice that the license files for Vehicle Spy are "out of date"

Step-by-step guide

The download location for the GM version of Vehicle Spy and all license files is...

www.intrepidcs.com/main/gm

Credentials are in Z:\Software Team\_LSIESystems.ini


Click the Latest Downloads link


If you're updating License Files:

Select the download for the GM Global NAO Corporate License file for Vehicle Spy

Use the previous credentials to open the zip file.

If the license files were actually updated, create a ticket for the patch_script to update the license files

The License Files DO NOT expire on the date listed, Intrepid has confirmed this multiple times
