BTSI Lab Systems Integration : Hotswap or Replacement PC Connectivity Issues

Created by Chelsea Flattery, last updated on Feb 21, 2024

Did you recently get a new PC and now you're seeing weird behavior? What on Earth is going on with this PC?Â 

When one PC is swapped out and a new PC is swapped in, we can see issues when the system has the hostname assigned to both PCs at the same time.

This issue presents as:

  • Not being able to remote into the PC with the hostname
  • Pinging the PC shows the wrong IP address
  • Grafana showing stale data or not showing data at all
  • iCentral showing the system as inactive even with iView and iTest running
  • The IT Store showing two PCs with the same hostname

Fix this issue by putting in an IT Store ticket for Technician Assistance and letting on-site IT know that the hostname hasn't resolved correctly in the system. On-Site IT will then reach out to someone in the depths of IT to fix the issue.

It will take a few hours for the system to correct this issue.
