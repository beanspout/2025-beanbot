BTSI Lab Systems Integration : How do I Install iTest and Other Programs on my Laptop

Created by Chelsea Flattery, last updated by Orbeen Alam on May 21, 2025

Because every laptop assigned to us has a different set of permissions, security features, and behaviors, the patch script is not designed to fully catch all edge cases (it's designed to work for TCWS lab machines).  Follow the attached guide to run all installs on a laptop, knowing there will be some variation from this as far as your particular laptop goes.

Step-by-step guide

  1. Request enhanced access

    Computer Enhanced Access (gm.com)

    Here’s what you need to enter, the first few fields will prepopulate with your laptop, if it does not populate select Searched and Asset Not Found.



  2. Select the reason for your request: Reason 3. Permanent elevated rights
  3. Level of enhanced computer access that will be given: Role: Persistent enhanced access

  4. The business justification should be something like (feel free to copy):

    The GBSL uses A&D Technologies iTest system for battery testing.  In order to create testing scripts, I need to install iTest 4.1 and its necessary component software on my GM issued laptop for script creation, debugging, and validation.  The scripts run by iTest require elevated permissions.

  5.  Follow the instructions in the attached Word Doc 
  6. Add your laptop to PCsList.ini, copy your entry from one of the other members of the LSIE team
