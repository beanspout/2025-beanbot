BTSI Lab Systems Integration : Moving Sentry to a new PC

Created by James Jezewski, last updated on Feb 10, 2021

The sentry PC is now http://usmpgtd9481gpp:9000/

The instance is running on docker on windows, and god help you if you ever need to update it.  It expects a full internet connection and doesn't like being behind a firewall.

First, the important folders:

  • `C:\Temp\onpremise-master` is where the docker config files are stored and will need to be migrated to the new PC.  Yes I know this is a terrible location, but I wasn't expecting the files to be needed outside the docker container.
  • `C:\Docker\DockerDesktop\DockerDesktop` This is where the docker VM is stored.
