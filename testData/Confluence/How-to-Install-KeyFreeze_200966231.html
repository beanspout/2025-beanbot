BTSI Lab Systems Integration : How to Install KeyFreeze

Created by Chelsea Flattery on May 31, 2023

To install the KeyFreeze application that will lock keyboard and mouse input after 30 minutes of inactivity, follow the process outlined below.

Step-by-step guide

  1. On your target machine, open up the Start Up folder by typing 'startup' into the File Explorer bar and pressing the Enter key. This will open the Startup folder where KeyFreeze will be placed.
  2. Navigate to Z:\_Manual_Installs\
  3. Copy the KeyFreeze folder
  4. Paste the entire KeyFreeze folder into C:\Program Files\ on the target PC and confirm the security prompt
  5. Open the C:\Program Files\KeyFreeze\ folder
  6. Right-click on KeyFreeze_x64.exe and select Create shortcut 
  7. Confirm the prompt to create the shortcut on the desktop
  8. Navigate to the Desktop and right-click KeyFreeze_x64 - Shortcut and click Rename, remove " - Shortcut"
  9. Drag this shortcut into the Startup folder you opened earlier
  10. Double-click the KeyFreeze shortcut in the Startup folder
  11. Confirm that the lock icon displays in the lower right of the screen
  12. Test the lock/unlock function


If the lock/unlock function does not work, verify that C:\Program Files\KeyFreeze\KeyFreeze.ini is configured correctly.  This should not have been edited without an announcement going out, so it is unlikely to have been changed without notice.
