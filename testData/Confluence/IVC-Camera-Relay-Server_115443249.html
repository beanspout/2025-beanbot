BTSI Lab Systems Integration : IVC Camera Relay Server

Created by Chelsea Flattery, last updated on Dec 13, 2023

For any administrative tasks, there is a singular admin account for the Relay Server.

Step-by-step guide

To make modifications to the Relay Server camera settings

  1. Log into the Relay Server as admin, you will need to login twice.
    1. Find the credentials in Z:\Software Team\_LSIESystems.ini
  2. Select the Management link at the top right of the page

To add a new camera

  1. Click on the Cameras tab
  2. Under the Basic heading, click Camera
  3. From the Camera drop-down select an available number and enter the camera name, based on location
  4. Click the Enable camera check box
  5. From the Encoder drop-down, select I-A1
  6. From the Pan/tilt drop-down, select 104
  7. From the Camera drop-down, select 104
  8. In the IP address field, enter the camera's IP address.Â  This should be the camera's SDF IP address, not the default IP provided by IVC
  9. In username and password, enter the default admin adminIVC username and password
  10. Image type is H.264
  11. Resolution Large, Quality High
  12. Set the FPS to 0
