BTSI Lab Systems Integration : Advanced Search

Created by Chelsea Flattery on Feb 03, 2020

As we migrate to Jira, remember that you can use JQL to advanced search.
Click 'View all issues and filters' in the upper right of the Issues screen to access the advanced search feature

Step-by-step guide

To access Advanced Search

  1. Select the Project (BTSI Common iTest Solution)
  2. Click Issues from the left-side toolbar
  3. Click View all issues and filters in the upper right-hand corner under your Profile photo
  4. Enter your JQL query
  5. Common queries can be saved


  6. Favorited searches will appear on the left toolbar of the Advanced Search screen
