BTSI Lab Systems Integration : Update NeoVI Firmware and Sync RTC

Created by Chelsea Flattery on Nov 23, 2021

This can be done with or without a licensed version of VehicleSpy.  
Step-by-step guide

For unlicensed versions of VehicleSpy:

  1. From the start menu select the IntrepidCS folder
  2. Select NeoVI Explorer
  3. Click Connect once the NeoVI device is highlighted
  4. If the firmware needs to be updated, it will automatically update. 
  5. Please allow time for the update to completely finish, the Firmware installer will open and close multiple times during this process. Wait until you see the 'neoVI FIRE has been reflashed' message appear before continuing.
  6. To update the Real Time Clock, click Synchronize RTC
  7. You will see the RTC update
  8. If you are troubleshooting the NeoVI device and need to manually reinstall device firmware, click Manual Reflash and see Step 5, above.

For licensed versions of VehicleSpy:

  1. You may either use the VehicleSpy software or the NeoVI Explorer
  2. From VehicleSpy, select the Setup menu
  3. Select Hardware
  4. Please wait for NeoVI Explorer to open
  5. You may now follow the step beginning at Step 3 from the previous section.
