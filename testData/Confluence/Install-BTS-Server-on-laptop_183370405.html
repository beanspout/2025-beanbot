BTSI Lab Systems Integration : Install BTS Server on laptop

Created by Collin Winans, last updated on Apr 12, 2024

I recommend installing BTS server/client on your laptop, it will allow you to experiment with tests/changes without affecting the production server in the lab. You can search for it in NI Package Manager, you wonâ€™t need a license for it.

You will need a license to run the BTS client, the LSIE team has a handful of licenses for development purposes. They are assigned to individual users, talk to Joe Juricic to get activated.

In NI License Manager, add the GM license server:


Once the software is installed, you will need to add the Lookup Table plugin for our application:

LookupTable.dllLookupTableInterface.lvlibp

  1. Put the LookupTableInterface.lvlibp in C:\Users\Public\Documents\National Instruments\Battery Test System\Compiler\Plugins (this is for the client)
  2. Put the Dll in C:\Program Files\National Instruments\Battery Test System\PackageLoader and C:\Program Files\National Instruments\Battery Test System\WebApplication (these are for the server)
  3. Restart the NI Battery Test Web Service
  4. Check the server logs C:\ProgramData\National Instruments\Battery Test System\SystemDefinitionWebAPI to make sure you see the Lookup Table plugin
  5. Install the client custom devices as explained here: SystemLink Custom Devices


There is also a config file we will need to modify to get the BTS server to use the Systemlink Server for login (Note: this might be different in the future if SystemLink migrates to a new server/version). Replace the file in C:\Program Files\National Instruments\Battery Test System\WebApplication\appsettings.json
