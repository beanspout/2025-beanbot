BTSI Lab Systems Integration : Request Visual Studio

Created by Gregory Thornton, last updated on Sep 08, 2022

This is for future reference if anyone needs Visual Studio to generate the .DLL file needed by the current BRFM Communication code. 

Step-by-step guide

  1. Go to https://galileo.gm.com, make sure to use MS Edge. Google Chrome can be a bit finicky when it comes to loading the website 

  2. Next Navigate to Services → Developer Tools → Development Tools then click on Development Tools

  3. Once here Fill out the form with the following data

    1. Project : 11000-Generic Project

    2. ASMS: 45856-Microsoft Development Platform

    3. Action: Request Licensed Software

      Make sure the action is a licensed software request otherwise Visual Studio will not be an option on the next menu

      Also be sure to add you GMID

  4. Select Visual Studio then click next

  5. Answer the questions with the following order:

    1. Click Submit.         

    2.  Be sure the version selected is Professional otherwise the ticket will be rejected

  6. Congrats! Now wait 5-7 days and you'll have Visual Studio
