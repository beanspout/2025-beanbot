BTSI Lab Systems Integration : My K8s Jenkins Merge Build Is Failing

Created by Constantin Balan on May 10, 2023

This article will be helpful if you are trying to merge two branches after you've resolved some conflicts, and you are getting unmerged file errors on K8s Jenkins. 



Relevant Link: [JENKINS-44598] git workspace should be reset before build PR merged with base branch - Jenkins Jira
First you should check to make sure there still aren't merge conflicts or unmerged files. If you've confirmed that there aren't, you should go into the 

configuration portion of your Jenkins pipeline. From there, you should add the "Clean before checkout" behavior. This will do a git clean and git reset which should resolve any issues 

you're having with this.
