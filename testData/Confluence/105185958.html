BTSI Lab Systems Integration : Add/Remove a user from Grafana

Created by Paul Rabahy, last updated by Chelsea Flattery on Aug 11, 2022

Stop

Before completing the process below, has the user entered a ticket into the service desk?  All users being added or removed must have a ticket for traceability with GMIT Cyber Security.  If the user is being removed, have their manager enter the ticket.



Go to https://admsvcs.gm.com/A2/ui/index.html#nav:collectiontab:collectiontabmanage

Click the proper group

All grafana-<role> groups pertain to the Estes Battery Lab, while the BCIC is indicated at the end of the role.

To Add a New User:
Click Add Group Member

Enter the user or users GMID(s) and click the search magnifying glass.

Click the plus next to the user's name to add them to the bottom pane.  If the user already has access, you will see the following message:

It takes about 24 hours to sync to https://grafana-bat.gm.com/.


To Remove an Existing User:
Enter the group that the user belongs to and click the - at the far right
