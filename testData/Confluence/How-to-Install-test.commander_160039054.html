BTSI Lab Systems Integration : How to Install test.commander

Created by Chelsea Flattery, last updated on Oct 12, 2022

Step-by-step guide

  1. Run Test Commander Installer from the _Manual_Installs folder, select all defaults for the installer
    1. (don’t do this next part if the PC is on the Bench, only if it’s at stand)
  2. Open Test.Commander

  3. Click License and enter:

    A&D Technology, Inc.

    A&D Technology, Inc.

    4379103090

  4. Open Wizard to start from scratch…

  5. Click Next
  6. Name it anything, I name it Test, click Next
  7. In the network drop-down, select the 192.168.1.99 Static option


  8. Select the second radio button to indicate that we are using a static IP-address, click Next
  9. Click Next, click Next
  10. Select the e.gate IP and click Next
  11. Click Next on the Change network setting page and wait while it sets up the parameters
  12. Click Next on the connect Slaves page and wait for it to scan
  13. Click Next on the Reading controller configuration page and wait for it to connect
  14. Click Next on the Checking for unique variable names page and wait for it to connect
  15. Click Next on the Finished successfully page
  16. Click OK on the View values window and close test commander
