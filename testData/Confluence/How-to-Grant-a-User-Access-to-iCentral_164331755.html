BTSI Lab Systems Integration : How to Grant a User Access to iCentral

Created by Chelsea Flattery on Nov 09, 2022

A user must submit a ticket to the Jira Service Desk for access to iCentral.  If they reached out to you directly instead of completing a ticket, redirect them to the following ticket link: https://tslm-jira.gm.com/servicedesk/customer/portal/7/create/388

Step-by-step guide

Beginning on Nov 17, 2022 all access is handled through Access Admin

  1. Navigate to Access Admin
  2. In the Collections tab all iCentral permission groups are displayed
  3. We control only the China and Estes groups, PMD is controlled by the Milford lab.
  4. Permissions groups are allocated as follows:

    China/Estes-Admin - Members of the LSIE team who have been trained on iCentral administration, Innovation Owners/Developers 

    China/Estes-Developer - An LSIE who isn't ready for admin

    China/Estes-Engineer - Any lab personnel who is not an LSIE

    China/Estes-Read-Only - Anyone else who has a strong business case to view data

  5. Enter the appropriate group for the new user
  6. Click the Add Group Members link
  7. Enter their GMID or first and last name in the search field

    NOTE: If you enter a GMID and a name, the GMID will be searched first.  Clear any GMIDs prior to searching for a user via first and last name
  8. Click the + sign next to the users name and then click Done


When a user is offboarded, they must be removed from the system via the - sign in the Group Members screen
