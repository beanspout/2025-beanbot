* Desired settings will be in {xxx} to stand out

** Need to edit the Module's Config.drv file and Solution's Config.drv file to get the changes to stick
** Will need to change the following in each of the Config.drv files, FOR CHAMBERS USING Modbus:
-Device ID (Ecc1 is 101, Ecc2 is 102, Ecc3 if needed is 103) * FOR ALL DRIVERS, INCLUDING GSD
-Modbus_X (Ecc1 is Modbus_1, Ecc2 is Modbus_2, Ecc3 if needed is Modbus_3)
-Port Number (Ecc1 is 2, Ecc2 is 6, Ecc3 if needed is 7) * PORT 8 IS USED FOR CALIBRATION ONLY
-Driver Name (match to Modbus_X)

*** If changing the physical port the particular chamber is connected to, you will need to make
sure the new COM Port settings match the old COM Port settings or per the Driver's port settings
