BTSI Lab Systems Integration : TestStand File Differ

Created by Collin Winans on Feb 26, 2025

General Command line use:

"C:/Program Files/National Instruments/TestStand 2021/Bin/FileDiffer.exe" "Path/To/File/1" "Path/To/File/2"


To run with git mergtools:

  1. Set up the mergetool in $HOME/.gitconfig

[difftool "TestStandFileDiffer"]

path = C:/Program Files/National Instruments/TestStand 2021/Bin/FileDiffer.exe

cmd = \"C:/Program Files/National Instruments/TestStand 2021/Bin/FileDiffer.exe\" \"$LOCAL\" \"$REMOTE\"

[mergetool "TestStandFileDiffer"]

path = C:/Program Files/National Instruments/TestStand 2021/Bin/FileDiffer.exe

cmd = \"C:/Program Files/National Instruments/TestStand 2021/Bin/FileDiffer.exe\" \"$BASE\" \"$LOCAL\" \"$REMOTE\" \"$MERGED\"

  1. During a merge with conflicts, run this command from the repo folder:

git mergetool --tool TestStandFileDiffer
