BTSI Lab Systems Integration : How to setup DigiBox and COM Ports

Created by Chelsea Flattery, last updated on Feb 19, 2024

Step-by-step guide

  1. Run the DigiBox Setup64.exe (don’t do this if the PC is on the Bench, only if it’s at stand and connected to the DigiBox)
    1. If 20.0.0.2 doesn’t show up:
      1. Check NIC, they’re probably swapped or not configured
      2. Set the IP Address/Subnet Mask for the “DigiBox” NIC to 20.0.0.1/255.255.255.0 and click Ok and Ok.
      3. Set the IP Address/Subnet Mask for the “IO Rack” NIC to 192.168.1.99/255.255.255.0 and click Ok and Ok.
    2. If they’re swapped
      1. Rename DigiBox to any random name or keyboard press
      2. Change IP to have 99 at the end to save time for later
      3. Rename IO Rack to DigiBox and change the IP to 20.0.0.1
      4. Rename the other one to IO Rack and change IP to 192.168.1.99
      5. Click Refresh on the Digi Setup
  2. If 20.0.0.2 shows up:

    1. Click Next
    2. Use the defaults, click Finish
  3. Right click on This PC (My Computer), select Manage
    1. Expand the Multi-port Serial Adapters
    2. Select PortServer TS 4 MEI (20.0.0.2)
    3. Right-click and select Properties
    4. In the Advanced tab, click Properties
    5. The ports need to be COM2, COM6, COM7, COM8
    6. If ports need to be edited:

      1. Start from the bottom and change them, if a port is already used you can’t reassign it
      2. Click the last port and click the Serial tab
      3. Change the port from the drop-down list
      4. Click the next port up from the bottom and do the same action.
      5. When all ports are correct, click Apply and Ok
  4. If a port is missing because it’s being used by a USB or other device:
    1. Expand the Ports (COM & LPT) menu
    2. Right-click on the port that is using the COM you need
    3. Select Properties > Port Settings > Advanced
    4. Change the port in the drop-down at the bottom of the Properties window


Turn off Power Management:

Right click and open the properties for each COM Port and USB connected device on the PC that is used for testing. Go to the Power Management tab and deselect the check box allowing the PC to put the device into power save mode.
