Mathematical Functions Table¶

Mathematic¶

Mathematic Functions

Function Summary Syntax
ABS Returns absolute value of a channel or value. 
ABS(channel)
inrange Returns True (1) if checkvalue is greater than or equal to lowvalue and less than or equal to highvalue, False (0) if it is not. This function supports the use of comma and space delimiters. 
inrange(lowvalue,checkvalue,highvalue)

inrange(lowvalue checkvalue highvalue)
int Returns truncated integer of a channel or value (i.e., drops decimal portion of the value). 
int(channel)
inv Returns inverse of a channel or value (i.e., reverses the value's sign). 
inv(channel)
max Returns the higher of two channels or values. 
max(channel1,channel2)
min Returns the lower of two channels or values. 
min(channel1,channel2)
mod Returns integer remainder of a division operation. 
mod(channel,divisor)
pct2value Returns a percentage of a value.
Formula: (percentage / 100) * source_value		 
pct2value(percentage,source_value)
ROUND Returns rounded integer of a channel or value. 
ROUND(channel)
sepdec Returns fractional portion of a channel or value (i.e., returns everything to the right of the decimal point). 
sepdec(channel)
sepint Returns integer portion of a channel or value (i.e., returns everything to the left of the decimal point). 
sepint(channel)
stddev Returns standard deviation of a channel over a period of time (in seconds). 
stddev(channel,seconds)
TRUNC Returns the integer of a channel or value that is less than or equal to the original value.
Example: TRUNC (3.2) returns 3, TRUNC (-3.2) returns -4		 
TRUNC(channel)
value2pct Returns what percentage channel1 is of channel2.
Formula: (channel1 / channel2) * 100		 
value2pct(channel1,channel2)

Thermocouple Conversion Functions Table¶

Thermocouple Conversion Functions

Function Summary Syntax
tc_j This function (and the 3 following it) allows a user to use an analog input voltage and cold junction compensation temperature to compute thermocouple temperature in degrees Celsius. 
tc_j(Ai1,CJC)
Ai1 is the voltage of the thermocouple in Volts, CJC is the temperature of the cold junction compensation in degrees C
tc_k K-type thermocouple. See description for the tc_j function. 
tc_k(Ai1,CJC)
tc_s S-type thermocouple. See description for the tc_j function. 
tc_s(Ai1,CJC)
tc_t T-type thermocouple. See description for the tc_j function. 
tc_t(Ai1,CJC)
