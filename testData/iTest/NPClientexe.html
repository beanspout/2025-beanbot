NPClientexe¶

This is a network performance diagnostic tool to help isolate network issues.
Usage:
NPClient <ipaddress> <send size> <receive size> [FAST]

The FAST option does not run on a timer, but spins to maximize throughput (rate will have more jitter).
Do not run this program while running iTest.
Example:
NPClient 10.102.226.152 32 32

If the update rate is not stable:

1. Try the wiggle test on each network connection to isolate poor connections
2. Try removing all other devices on the network except the real-time PC and the GUI PC
3. Try a direct connection without a switch

Example:
C:\ADT\iTest4.5\Execute>NPClient.exe 10.102.226.152 32 32
Connect Tx:32 bytes Rx:32
Server says:OK
Send:32.032 k/sec Receive:32.032 k/sec Rate:1001.0 Hz (1001.0 max,988.0 min) MaxLatency:0.016s Errors:0
