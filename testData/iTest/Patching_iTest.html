Overview¶

A&D Technology relies on a series of patches to add new features and address issues between version releases. Official patches are available on a quarterly basis for supported versions of iTest. These patches can be accessed on the A&D Technology customer service portal. The customer portal is available to all warranty and Safety Belt customers and can be accessed via https://support.aanddtech.com.

NOTE: The patch process may require automatically updating the Hasp software using HaspUpdate.bat. This requires elevated (i.e., administrator) permissions when patching. If these permissions are not available and a patch for the Hasp software is present, an error message will display (unless running silently), indicating that HaspUpdate.bat must be manually ran as an administrator. The $SYSTEMDIR\Patches\Patches.log file is updated with the error message.

Patching Process¶

To patch iTest, do the following:

WARNING: Reverting to a previous iTest build is not supported by this process. Contact A&D Technology for assistance.
  1. Run iTestXXSetup.exe for the version you would like to patch, where XX is the product version (e.g., iTest44Setup.exe).
     
    Setup

     
  2. Click Next through the automatically populated settings. For more information, refer to the Standard iTest Installation Guide.
    • Install iTest to the same folder as your previous version.
  3. Click Finish to run the new version of iTest.
