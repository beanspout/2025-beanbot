Release Notes

The Release Notes section contains detailed "What's New?" information for the most current version of iTest.
 

Standard iTest Installation Guide

The Standard iTest Installation Guide documents the software installation process, such as minimum system requirements, licensing utilities, and installation steps.
 

Getting Started Guide

The Getting Started Guide discusses key terms and concepts used throughout A&D applications and documentation.
 

iTest Console Operator's Guide

The iTest Console Operator's Guide discusses the information required to use a configured iTest application. This document provides the operator with common run-time interfaces and necessary instructions for successful use of iTest when performing tests.
 

AutomationPanel Operator's Guide

The AutomationPanel Operator's Guide provides instruction and direction for using AutomationPanel to develop user interfaces. This guide is only to be used for applications delivered by A&D Technology to support AutomationPanel.
 

iTest Application Development Guide

The iTest Application Development Guide covers several basic and advanced topics such as calculations, control loops, channels, data logging, limits, and VCL. The advanced topics are used infrequently or are programmed by A&D Technology for customer specific needs.
 

Developer Tools

The Developer Tools guide details how to use SolutionBuilder and Test Manager to manage various aspects of your solution. In additional, several debugging methods and tools are included in this guide.
 

How-To Guides

The How-To guide documentation consists of tutorials that walk you through specific tasks in iTest. Topics range from beginner to intermediate level tasks and advanced tasks.
 

Reference Lists

The Reference Lists section includes tables with descriptions of INI options, mailslot messages, math functions, field definitions, etc.
 

Standalone Tools

The Standalone Tools documentation discusses iTest executables that are designed to perform specific functions. Tools that were not previously covered in the documentation are listed in this section.
 

iTest Maintenance

The iTest Maintenance documentation discusses troubleshooting topics and system updates, such as updating your dongle, running a problem report, auto-patching iTest, and using advanced methods for updating your system.
 

iTest Demo Applications

The iTest Demo Applications section contains write-ups for demo solutions and modules that are included with a standard iTest install.
 

Third-party Software Credits

The third-party software credits section lists software licenses for products A&D Technology utilizes in iTest products.
