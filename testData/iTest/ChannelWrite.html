ChannelWrite Overview¶

The ChannelWrite utility application can be used to set non-string channel values when iTest is running. It is intended to be used with the iTestWD utility to help bring the system to a safe condition if a catastrophic failure occurs in iTest. However, it can also be used as an easy interface for setting RDB channel values from other languages that require a low speed update of the RDB. For example, this tool could be run from a batch file.

This utility displays no visible output.

NOTE: It is not possible to use this tool to set string values.

Execution¶

The utility is simply called from a command line with the following arguments:

ChannelWrite iTestChannel value

Example¶

ChannelWrite Throttle 0.0
