The powertek.ini file options are used to configure the solution, modify existing base functionality, or add new functionality in order to provide additional flexibility for core functions of the iTest software. These options are described in the tables below and are grouped based on relevancy. For example, any powertek.ini settings relevant to iTest.BTS will be listed in the iTest.BTS section.

The tables below include the Version, GUI, and Application Interface (if applicable) columns to provide the following additional information:

  • Version: The product version the powertek.ini option was first introduced.
  • GUI: This indicates whether the powertek.ini option can be edited in Test Manager's Global Settings editor.
  • Application Interface: The applications that are affected by the powertek.ini option (e.g., iTest Console, AutomationPanel).
