SendToMailslot.exe¶

The SendToMailslot.exe sends mailslot messages to a driver or any other executable. Messages can be a command, procedure, or note to be communicated.

This is a command line executable; there is not a GUI associated with it.

Syntax

SendToMailslot.exe <mailslot-addr> <message>

Parameters

message: The message to be communicated.

mailslot-addr: The location in which the message will be sent. The address can be fully qualified or abbreviated:
\\.\mailslot\SoftPanel or Softpanel

Examples

SendToMailslot.exe Softpanel “Hello”
