INtime Extensions¶

The version of the INtime TR-OS you are running determines the .rta file to launch.

INtime Version and File Extension

INtime Version File Extension
3.13 <filename>.rta OR <filename>_rt.rta
4.2 <filename>_rt4x.rta
5.2 <filename_rt5x.rta

Example:

To run the CalcEngine utility on an INTime 4.2 system, you would use the CalcEngine_rt4x.rta file in the RTXPreload powertek.ini setting.

 RTXPreload=\\CalcEngine_rt4x.rta>>

On an INtime 3.13 system, you would use the CalcEngine.rta file instead.

 RTXPreload=\\CalcEngine.rta>>

NOTE: In some iTest GUI editors, there may only be an option to select a file ending in a file extension that doesn't match what should be selected based on the above table. In these instances, select the option that ends with .rta and iTest will launch the correct INtime executable file for you.
