Overview¶

The Video panel enables you to view videos and control during runtime in AutomationPanel. Video playback is controlled by using start, stop, and other buttons that you add to the layout. The Video panel supports .mp4 files, which can be displayed at the rate of 0.5, 1, or 2.

This document details how to set up your solution to use the Video panel.

Creating Channels¶

The first step is to create the channels used to display information, such as duration and position of the video.

For the example in this guide, the following channels will be created and used: vidDuration and vidPosition. To create these channels, do the following:

The first step is to create the channels used to display information, such as duration and position of the video. For the purposes of this guide, the vidDuration and vidPosition channels will be created and used.

To create these channels, do the following:

  1. Launch Test Manager or SolutionBuilder and navigate to the Channel Definitions editor.
  2. Right-click in the editor and select the Add to end option. This opens the New Channel dialog.
     
    New Channel Dialog

     
  3. For both channels, use the Virtual Output channel type.
  4. Enter vidDuration/vidPosition in the Channel Name field.
  5. Save your changes and exit the application.

Adding the Video Panel¶

The next step is to add the Video panel to your layout. You can use AutomationPanel or SolutionBuilder to add this panel.

To add the Video panel in AutomationPanel, do the following:

  1. In an existing or new layout, right-click the layout and select the New Panel... option. Alternatively, select File > New > Panel from the menu.
  2. Select the Video panel in the Add New Panel dialog.
  3. Click OK.

To add the Video panel in SolutionBuilder, do the following:

  1. Navigate to the Layouts editor.
  2. In an existing or new layout, right-click in the editor and select the New Panel... option.
  3. Select the Video panel in the Add New Panel dialog.
  4. Click OK.

Configuring Video Panel Properties¶

Once you have added the Video panel to your layout, configure the following properties:

Video Panel Properties

Property Configuration
Automation ID This value is used to modify panel properties, such as to start and stop the video at runtime, using channels or mailslot messages (e.g., Video).
File Path Click the button to browse for a video file to play in the panel. Alternatively, enter the full filepath in this field. Note: Only .mp4 files are supported.
Position Channel Name This channel is updated with the current position of the video while it is playing.

Click Save once finished.

Setting Up Control Buttons¶

After configuring the Video panel, you can add a Tile panel with buttons that will enable you to control the progression of the video in your layout. In this example, eight buttons will be created to perform the following actions:

  • Start Video
  • Stop Video
  • Set Position to Beginning
  • Set Rate to 0.5
  • Set Rate to 1
  • Set Rate to 5
  • Get Position
  • Get Duration

Adding the Buttons and Channel Tiles¶

To set up your control buttons, do the following:

  1. In your layout, add the Tile panel. Right-click in the Tile panel and select Add > Button.
    Image
  2. Set the Button Type property to Push. Enter the label for the button in the Text Off setting.
     
    Text Off Label

     
  3. Repeat for each button until you have eight.
  4. Right-click in the Tile panel and select Add > Channel.
  5. Set the Channel property to vidPosition and the Label property to Video Position.
  6. Repeat on the second channel tile for vidDuration (Video Duration).
  7. Arrange the buttons and channel tiles using the Move Previous and Move Next options.
    After you've added your push buttons, add two channel tiles to the same panel to display the video duration and video position when you press Get Duration and Get Position.

Setting Up Mailslot Messages¶

Mailslot messages enable you to perform an action or send a message when you click a button. You must configure the buttons in the Tile panel to send message to start, stop, and control the video progression. The $SetItemProperty and GetItemProperty mailslot messages are used to control the Video panel.

To set up mailslot messages, do the following:

  1. Open the properties for each button.
  2. Navigate to the Message Off property and click the button. This opens the MessagesEditor dialog.
     
    MessagesEditor

     
  3. Click the Destination drop-down and select \\.\mailslot\Softpanel.
     
    Mailslot Destination

     
  4. Enter the following mailslot messages for each button:
    Button Maislot Message
    Start Video \\mailslot\Softpanel>>$SetItemProperty;Video.Play=1
    Stop Video \\mailslot\Softpanel>>$SetItemProperty;Video.Play=0
    Set Position to Beginning \\mailslot\Softpanel>>$SetItemProperty;Video.Position=0
    Set Rate to 0.5 \\mailslot\Softpanel>>$SetItemProperty;Video.Rate=0.5
    Set Rate to 1 \\mailslot\Softpanel>>$SetItemProperty;Video.Rate=1
    Set Rate to 5 \\mailslot\Softpanel>>$SetItemProperty;Video.Rate=5
    Get Position \\mailslot\Softpanel>>$SetItemProperty;Video.Position=vidPosition
    Get Duration \\mailslot\Softpanel>>$SetItemProperty;Video.Duration=vidDuration
  5. Click Save when finished.
