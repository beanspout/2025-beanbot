Overview¶

DspDrv.exe is an application that enables you to open iTest Console displays, which is useful for touch screen devices. DspDrv.exe can open dialog, standard, and right displays independently of iTest Console when you launch iTest.exe with the GUI powertek.ini setting. It can also be used to open custom and right displays with Softpanel mailslot messages.

NOTE: This application runs in the background; there is no interface.

Using the powertek.ini Option¶

When iTest.exe is launched, the GUI powertek.ini option opens a dialog, standard, or right display by invoking DspDrv.exe.

Example:

In this example, DspDrv.exe will launch the Hsdl1 custom display when iTest.exe is launched.

GUI=dspdrv.exe $SUPPORTDIR\EXECUTE\CUSTOM\Hsdl1.dsp

 

Using Mailslot Messages¶

$CUSTOMDISPLAY¶

The $CUSTOMDISPLAY Softpanel mailslot message opens a custom display by invoking DspDrv.exe.

Example:

In this example, DspDrv.exe will open the EngineSettings custom display (EngineSettings.dsp).

$CUSTOMDISPLAY;EngineSettings

$RIGHTDISPLAY¶

The $RIGHTDISPLAY Softpanel mailslot message opens a right display by invoking DspDrv.exe.

Example:

In this example, DspDrv.exe will open the RightSettings right display (RightSettings.dsp).

$RIGHTDISPLAY;RightSettings

 

Using the DspDrv.exe Mailslot¶

When you use DspDrv.exe, a mailslot is opened with the AutomationPanel layout or iTest Console display name. These mailslots can be used to send messages, such as MINIMIZE, MAXIMIZE, and CLOSE. Refer to the Messages section below for a complete list of supported mailslot messages.

Syntax:

\\.\mailslot\<DisplayName>

Where:

<DisplayName> refers to the name of the open iTest Console display.

Example:

\\.\mailslot\DataLogViewerLayout

Messages¶

MINIMIZE¶

Minimizes the display. This message is case sensitive.

MAXIMIZE¶

Maximizes the display to the size of the screen.

RESTORE¶

Restores the display to its previous size and location if it is currently minimized or maximized. This message is case sensitive.

DefaultLocation¶

Returns a display to the size and location that were saved in the display file when it was created. Also Unminimizes or unmaximizes it. This message is case sensitive.

ShowWindow¶

Moves the display above any other displays or windows that may be on top of it. This message is case sensitive.

CLOSE¶

Closes the display.

$PLOTSTART¶

Starts adding data to a plot. This affects all plots on a display.

$PLOTSTOP¶

Stops adding data to a plot. This affects all plots on a display

$PLOTCLEAR¶

Removes any existing data currently displayed in a plot (stopped or not stopped). This affects all plots on a display.

$PRINT¶

Displays the print dialog to print the display.

$REFRESH¶

Refreshes the display by rereading configurations from the real-time database. This should be used after reloading AKA files if a display is using customer names instead of channel names.

$SETARRAYINDEX <index>¶

Specifies what index to use to display array channels. This can be used if no index was specified with the array channel when the display was created. The value for <index> should be a number.

Example:

$SETARRAYINDEX 1
$SETARRAYINDEX 4
