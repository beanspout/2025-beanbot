Overview¶

The iTest package includes the PythonWin IDE to help with debugging Python scripts. You can launch this as a standalone application or from Test Manager by right-clicking a script and selecting the Debug... option. In some cases, the IDE may not be adequate for all developers. This guide explains how to install Eclipse and connect it to the iTest PythonEngine executable to use the Eclipse debugger instead of PythonWin. It outlines the process for downloading and installing Eclipse and adding PyDev to enhance your debugging system.

Downloading and Installing Eclipse¶

To download Eclipse:

  1. Using your web browser, navigate to https://www.eclipse.org/downloads/. At the time of this document's creation, Eclipse Neon is the version used.
     
    Get Eclipse Neon

     
  2. Click Download 64 bit. This opens a new web page.
  3. Click Download on the web page to download the program.
     
    Download

     
  4. Save the Eclipse install to a location on your PC.
  5. Open the install. A list of Eclipse installations displays.
     
    Eclipse Installer

     
    Installation Type

     
  6. Select Eclipse IDE for C/C++ Developers.
  7. Select an installation folder.
     
    Installation Folder

     
  8. Click Install to begin the program installation.
     
    Installing

     
  9. Click Launch to launch the program.
     
    Launch

     
    eclipse neon.1

     
  10. Select a workspace directory and click OK. The Welcome screen displays.
     
    Select Your Workspace Directory

     
    Welcome

     

Installing PyDev¶

To install PyDev onto Eclipse:

  1. Select Help > Install New Software from the main menu. The Available Software dialog displays.
     
    Install New Software

     
    Available Software Dialog

     
  2. In the Work with field, enter http://pydev.org/updates and click Add.
  3. You are prompted to add a repository. Enter PyDev and click OK.
     
    PyDev Repository

     
  4. Click the Select All button, and then click Next.
     
    Select All Installation Items

     
  5. Review the installation items and click Next.
     
    Review Installation Items

     
  6. Review the licenses and accept the terms of agreement. Click Finish.
     
    Review Licenses and Accept Terms of Agreement

     
  7. Click OK on the security warning dialog.
     
    Security Warning Dialog

     
  8. Select the option to trust the certificates and click OK.
     
    Trust Certificates

     
  9. Once PyDev has been installed, you will need to restart Eclipse. When prompted, click Yes. Eclipse will close and reopen.
     
    Restart Eclipse

     
  10. Select Window > Preferences from the main menu to open the Preferences dialog.
     
    Preferences

     
    Preferences Dialog

     
  11. Use the navigation menu to open the PyDev, Interpreters, and the Python Interpreter branches.
     
    Python Interpreter Branch

     
  12. Click the New button. This opens the Select Interpreter dialog.
     
    Select Interpreter Dialog

     
  13. Enter iTestPython in the interpreter name field.
  14. Use the browse button to search for the Python interpreter located in the iTest installation. This is the python.exe application file.
     
    iTest Installation Python Interpreter

     
  15. Click Open.
     
    Python Interpreters

     
  16. Click OK. A new dialog displays, prompting you to select folders to add to the SYSTEM python path. By default, all folder options are selected. Use this setting.
     
    Python Folder Selection

     
  17. Click OK. Your Python Interpreter set up is complete.
     
    Python Interpreters

     
  18. Select OK to exit the Preferences dialog.

Creating and Setting Up Your Python Project¶

To create a new Python project:

  1. Exit the Welcome screen.
  2. Open the PyDev perspective. Select Window > Perspective > Open Perspective > Other... from the main menu. This opens the Open Perspective dialog.
     
    PyDev Perspective

     
    Open Perspective

     
  3. Highlight PyDev and click OK.
     
    New PyDev Perspective

     
  4. Right-click the default named folder and select New. The Select a Wizard dialog opens.
     
    Select a Wizard Dialog

     
  5. Expand the PyDev folder.
  6. Select the PyDev Project option.
  7. Click Next. This opens the PyDev Project dialog.
     
    PyDev Project Dialog

     
  8. Enter "iTest" as the name of the project.
  9. Use the remaining default settings. Click Finish.
  10. Right-click the iTest project folder and New > Other....
     
    Your Project

     
  11. Expand the PyDev folder and highlight the &Link to Existing Source subfolder.
     
    Link to Existing Source

     
  12. Click Next. This opens the Add Pre-Existing Source Folders dialog.
     
    Pre-Existing Source Folders Dialog

     
  13. Enter "PythonScripts" in the Name field.
  14. Click the Browse button to locate the installation location of your PythonScripts folder.
     
    Link to Existing Source

     
  15. Click Finish.

Setting Up Debug Configuration¶

To set up debug configuration:

  1. Expand the iTest project folder and the PythonScripts folder.
     
    Expand the Project Folder

     
  2. Right-click the script to debug. Select Debug As > Python Run from the menu options.
     
    Debug Script

     
    The debug of the python script results in an error message. This is due to missing DLLs.
     
    Console Message

     
    Open the Eclipse Debug Configurations:
     
  3. Select Run > Debug Configurations... from the main menu. This opens the Debug Configurations dialog.
     
    Debug Configurations

     
    Debug Configurations Dialog

     
  4. Select your configuration.
     
    Configuration

     
  5. Select the Environment tab.
     
    Environment Tab

     
  6. Select the New button. This opens the New Environment Variable dialog.
     
    New Environment Variable

     
  7. Add the iTest Execute folder to the path.
     
    Adding the iTest Execute Folder

     
  8. Click OK.
  9. Click Close and save your new debug configuration.
     
    You can now begin debugging your iTest python scripts.

Debugging iTest Python Scripts Remotely¶

To debug iTest Python Scripts remotely:

  1. Open the Python Debug Perspective. Select Window > Perspective > Open Perspective > Other... from the main menu.
     
    Opening Debug Perspective

     
  2. Select Debug from the Open Perspective dialog.
     
    Open Perspective Dialog

     
  3. Click OK.
  4. Select the PyDev Debug Server icon.
     
    Starting the PyDev Server

     
    Now, when you call Python Scripts from iTest, it will stop at the breakpoint.
