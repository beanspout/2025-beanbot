Overview¶

iTest is integrated with the Python programming language to provide the system with an additional layer of automation. This feature supplements existing VCL code, enabling users to automate their own systems with a common and contemporary coding language. You can use SolutionBuilder or Test Manager to quickly and efficiently add and debug Python scripts as well as create procedures that can be used during runtime. Python scripts ( .py files) written and saved in SolutionBuilder/Test Manager are encoded in UTF-8.

Python Restrictions¶

Python scripts written and used within the iTest environment do not support sys.argy. In iTest, scripts are executed from VCL and mailslot messages.
