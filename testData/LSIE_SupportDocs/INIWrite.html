Overview¶

INIWrite is a Windows application that updates the value of a keyword=value pair in an .INI file. If the provided keyword is a valid channel in iTest and is found in the file, the value of that keyword=value pair in the file is updated to reflect the current value of the iTest channel. The utility will update all found instances of the keyword=value pair in the .INI file. If a provided keyword is not found in the file, then the new keyword=value pair is added to the end of the file.

Command Line¶

 INIWrite.exe <Channel Name> [File Name]

Parameters¶

<Channel Name>: The name of the iTest channel that is searched for in the target .INI file as the keyword portion of a keyword=value pair. If a customer name is given for this parameter, then the target .INI file will be searched for a keyword that matches what was given. Hence, if a customer name is provided, but the file uses the channel name as the keyword, a new keyword=value pair will be added to the end of the .INI file with the value of the iTest channel that is assigned to the customer name that was given.

This utility supports the use of virtual array channels. When working with virtual array channels, the array index can be included as part of the channel name (e.g., voaDemoArray[1]). If the array index is included, then the INIWrite utility will search the target file for that exact keyword=value pair including the array index and update the value based on the given array index. If an array channel is not indexed on the command line, then the non-indexed keyword in the file will be updated with the value from the channel's [0] array index.

<File Name>: Optional parameter. The name of the file that is updated when INIWrite is executed. The file name must include the file extension. If the file name is not provided, the default file is the Master.ini file. If a fully qualified file path is not provided but a filename is provided, the file path defaults to the $SUPPORTDIR folder.

Examples¶

 Command Line Entry:  INIWrite.exe voaDemoArray Master.ini
* Will update the voaDemoArray keyword in the $SUPPORTDIR\Master.ini file with the voaDemoArray[0] value. 
 Command Line Entry:  INIWrite.exe voaDemoArray[1] C:\ADT\iTest3.7\Sys.ini
* Will update the voaDemoArray[1] keyword in the C:\ADT\iTest3.7\Sys.ini file with the voaDemoArray[1] value.
