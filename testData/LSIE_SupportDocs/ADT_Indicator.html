Overview¶

The ADT Indicator display object will display message strings and/or color bars that change depending upon the value of an associated database channel.

ADT Indicator

 

Field Definitions¶

General Tab¶

General Tab

Option Description
Channel Name Click the Channel... button to open the Channel Selection dialog, which you can use to select the database channel whose name and value will display on the ADT Indicator. Note that selecting a virtual string channel will have an effect on later settings.
Edge Select a border for the ADT Indicator.
Indicator Type Select whether the indicator is a string or color indicator, both a string and color indicator, or displays an image.
Value, String, Color, Image Path Where Chan ≤ ValA / StrA + ColA, ValA < Chan ≤ ValB / Str + Col B, ValB < Chan ≤ ValC / Str + Col C, and ValC < Chan / "????" string + gray color.
You can also display a different image in the Image Path based on the value of a channel.
If a Virtual String channel has been defined, any mapping settings in terms of string or color will be ignored and only the channel name and the value will be displayed.
Add New Row Adds a new row to the Value, String, Color table to specify additional Value, String, Color combinations.
Delete Row Deletes an existing row.
Label Field (located to the left of Hide Label) Select a label for the control (Automatic, Customer Name, or Module Name) as defined for the database channel.
Hide Label When selected, a label showing the channel (Customer Name, Module Name) will not display in the ADT Indicator.
Show Border When selected, adds borders around the ADT Indicator control.
Control ID Provides a unique name for the control. This name can be used in mailslot messages to change control properties during runtime on the iTest Console.
Permit Runtime Changes When selected, allows changes to be made programmatically.

 

ADT Indicator Object Properties¶

These properties are useful when issuing the following mailslot message:

\\.\mailslot\DisplayName>>$SETITEMPROPERTY ControlID PropertyName PropertyValue

The following are the names of the properties that can be changed by mailslot message to the display. These property names may or may not be the same as the property names shown on the display object property setup dialogs. Most of these property names are self-explanatory. The first column is the data type of the property.

The int or OLE_COLOR for color is the color RGB values. “BSTR” means the type of the property is string. “int”, “float”, “long”, “short”, “double” means the type of the property is number. “Boolean” means the property value should be 1 or 0.

Font property syntax is complicated.
<font name>,<bold>,,<italic>,<font size>,,<strikeout>,<underline>,
example: Arial,0,0,0,97500,0,0,0,400
