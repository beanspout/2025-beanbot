Overview¶

These functions are intended to be used with array channels. Since array channels start with index 0, all of these functions will assume that the index arguments are 0 based.

ARRAYCOPY¶

Copies the elements of one array channel to another. Arrays must be of the same type, otherwise you will get a compile error. Elements from the source array are copied to the destination array. When copying, the size of the smallest array is used. For example, if the source is 5 elements and the destination is 10 elements, 5 elements will be copied; if the source is 10 elements and the destination is 5 elements, 5 elements will be copied.

Syntax:

ARRAYCOPY <DestinationArray> <SourceArray>

Parameters:

DestinationArray: The name of the array channel of the destination array.

SourceArray: The name of the array channel of the source array.

Example:

ARRAYCOPY myarray1 myarray2

 

ARRAYINIT¶

Initializes the elements of an array channel to a known value. If the expression type doesn't match the array type, you will get a compile error.

Syntax:

ARRAYINIT <ArrayChannel> <Expression>

Parameters:

ArrayChannel: The name of the array channel.

Expression: The expression of value that each element in the array is set to.

Example:

ARRAYINIT myarray 1 + 2
ARRAYINIT mystrarray "itworks"

 

ArrayItemIndex¶

Returns the array index of the first found instance of the specified value in the array. Returns -1 if the value is not found. This function supports both numeric and string values. When searching for numeric values, the function uses an approximate value check that supports up to 14 digits of precision.

Syntax:

ArrayItemIndex(<ArrayChannel>, <ArrayValue>)

Parameters:

ArrayChannel: The name of the virtual array channel.

ArrayValue: The value in the array that is used to look up the array index. This parameter can also be a channel name, customer name, indexed virtual array channel, or local variable.

Example:

SET voReturnChannel ArrayItemIndex(voArraySpeed, 75)

 

ArrayMax¶

Finds the maximum value between two indexes in an array.

Syntax:

ArrayMax(<ArrayChannel>, <StartIndex>, <StopIndex>)

Parameters:

ArrayChannel: The name of the array channel.

StartIndex: The first index of the array subset.

StopIndex: The last index of the array subset.

Example:

SET voReturnChannel ArrayMax(voArrayTemp, 5, 180)

Notes:

  • If <StartIndex> is < 0, it will use 0.
  • If <StartIndex> is >= <StopIndex>, then the function will return -999.
  • If <StopIndex> is < 0 or it is >= the size of the array, it will use the size of the array minus 1.
