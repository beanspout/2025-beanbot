Overview¶

INIWriteFileFast is a Windows application that updates the value of any keyword=value pairs listed in an .INI file. INIWriteFileFast is similar to the INIWrite tool; however, all of the keyword=value pairs listed in an .INI file are updated to reflect the current iTest channel values if the keywords in the file match channel names in the RDB. The use of customer names and virtual array channels in the .INI file is supported.

When working with virtual array channels, if the array index is included as part of the keyword listed in the .INI file, then the value will be updated with the value based on the given array index. If a non-indexed array channel is listed and the aryIDx= parameter is not used, then the value will be updated with the value from the channel's [0] array index.

Command Line¶

 INIWriteFileFast <File Name> [aryIDx=n]

Parameters¶

<File Name>: The name of the file that is to be updated by this utility. It must include the fully qualified filepath and the file extension. Use of a macro, like $SUPPORTDIR, is supported. If a filepath is not provided, then this utility will default to looking for the file in the current working directory.

<aryIDx=n>: Optional parameter. If any keywords in the file matches the name of an unindexed virtual array channel, then the value 'n' from this parameter will be used as the array index (element) of the channel when updating the value.
